# Web Application Server

### How to start?
> setting by virtual env
```
python3 -m venv virtual
```

> enable virtual env 
```
source activate
```

> lib install
```
pip3 install -r requirments.txt
```

> run server
```
cd Opendata && python3 manage.py runserver
```
